# Progetto Web: Gruppo Caviro - Sostenibilità ed Economia Circolare

Questo progetto è stato sviluppato come elaborato d'esame. Consiste in una piattaforma web interattiva (Front-end in HTML5, CSS3/Bootstrap, JS) supportata da un server locale (Back-end in Node.js/Express) per la gestione sicura di un form di Lead Generation e il salvataggio asincrono dei dati su file CSV.

## Istruzioni per l'avvio locale (per la Commissione)

Per testare l'applicativo sul proprio terminale locale, seguire questi passaggi:

1. Assicurarsi di avere **Node.js** installato sul proprio computer.
2. Aprire il terminale nella cartella principale del progetto.
3. Installare le dipendenze necessarie (Express) digitando il comando:
   \`npm install express\`
4. Avviare il server locale digitando:
   \`node server.js\`
5. Aprire il browser web e navigare all'indirizzo: **http://localhost:3000**
